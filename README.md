# Rangabhoomi - Ticket Booking Platform

A full-stack ticket booking platform built with Django and React.

## Setup Instructions

### 1. Backend (Django)
Ensure you have Python installed.
```bash
pip install -r requirements.txt
```

### 2. Frontend (React)
Ensure you have Node.js installed. Navigate to the `frontend` folder:
```bash
cd frontend
npm install
```

### 3. Running the Project
To start both the Django backend API and the React frontend simultaneously, go to the top-level directory and run:
```bash
.\start_servers.bat
```

The React App will be available at `http://localhost:3000/`.
The Django API operates internally on `0.0.0.0:8000`.
